Projects

1) Scrolling Message Display Using Microcontroller (8051).

A Display unit that takes the input (to be displayed on the LED Screen) from the user, using the Serial Port and Displays as a Scrolling Message on the LED Matrix Display by using “8051 Microcontroller and Embedded C programming”.

2) Braille Script to Voice Conversion.

A Wearable Device that helps the Blind and Dumb people to Communicate with Voice by giving the input to the developed 6 Buttons Braille Keyboard, which is processed through Arduino and gives the output as Voice through Speaker.

3) Quadcopter with Arial Surveillance.

A Quadcopter built on the platform of ArduPilot Mega 2.8, capable of transmitting the video from the camera attached to it to the Mission Planner Software at the Base Station, which can be stored and analyzed. It has a Capability of carrying the weight of 5KG.

4) Analysis of PAPR in OFDM.

OFDM (Orthogonal Frequency Division Multiplexing) a multi carrier modulation technique with high PAPR (Peak to Average Power Ratio), which is analyzed in different modulation techniques like BPSK, QPSK, QAM, APSK using Binary and Gray Data in MATLAB.

5) Line Maze Solver Bot Using Arduino.

An Arduino based Line Follower Bot, further Programmed to solve the “Line Maze” with the shortest path possible using the method “Left Hand Rule”.

6) Profile Webpage.

A Profile Webpage designed using HTML5, CSS3 for myself in free time. The Page you are Viewing now.
